Spatial Subdivision

Spatial subdivision was implemented for an acceleration mechanism. A bounding box of the scene is subdivided into n equally sized cells. Each ray walks through this grid testing intersection only with objects found in the current grid cell.

Orignial: 2m56s
Fastest: 2.5s
Improvement: 88 times faster!
